## Permission Fields Structure

Permission fields contain a nested object structure mapping role IDs to permission arrays:

```json
"permissions": {
  "role_id": [
    "permission_name",
    "another_permission"
  ]
}
```

### Common Permission Types

| Permission | Description |
|------------|-------------|
| `CREATE` | Create new documents |
| `UPDATE` | Modify existing documents |
| `DELETE` | Remove documents |
| `ACTOR` | Access to Actor documents |
| `ITEM` | Access to Item documents |
| `COMBAT` | Access to Combat documents |
